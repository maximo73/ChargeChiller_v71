 ChargeChiller_v7 변경

1. 팬 제어 로직 변경
   기존: 전원이 투입되면 자동운전 신호입력 유무에 상관없이 온도에 따라 팬이 ON/OFF 제어됨.
   변경: 자동운전 신호입력이 있는 경우에만 온도에 따라 팬이 ON/OFF 제어됨. 

2. 펌프 수동운전 추가
   1) 자동운전이 아닌 조건의 메인화면에서 Up Key + Enter Key를 누르면 설정된 속도로 운전하며 수동운전모드가 됩니다.

   2) 수동운전모드로 운전중일 때에 메인화면과 같이 압력과 온도를 Up 혹은 Down Key를 눌러 각각 확인이 가능합니다. 

   3) 수동운전모드에서 Enter Key를 누르면 수동운전 중에 속도를 설정할 수 있습니다.
      속도 설정화면에서는 속도를 10% 단위로 0(10%)~9(100%) 범위에서 Up Key 혹은 Down Key를 눌러서 설정할 수 있습니다.
      Enter Key를 누르면 속도 설정화면에서 압력/온도 정보화면으로 전환됩니다. 

   4) 수동운전모드에서 속도설정화면이나  Mode Key를 누르면 속도가 감속정지하고 수동운전모드가 종료됩니다.

   5) 수동운전모드에서는 외부에서 자동운전 신호를 입력해도 동작하지 않습니다.
